# Wireframes – Sprint 1

Este diretório contém os protótipos das telas desenvolvidas para a Sprint 1 do sistema de gestão acadêmica.

As telas foram projetadas com foco na usabilidade para os dois perfis principais, garantindo uma navegação intuitiva, eficiente e alinhada às necessidades dos **Professor** e **Coordenador**.
